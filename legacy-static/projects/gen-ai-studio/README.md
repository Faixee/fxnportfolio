# Gen AI Studio - Enterprise Edition

A professional Generative AI content creation suite mimicking GPT-4 capabilities with token usage analytics and latency tracking.

## Features
- **Multi-Modal Simulation**: Text generation with style and length controls.
- **Analytics Dashboard**: Real-time token counting and latency metrics.
- **Role-Based Access**: Mock user profile and plan management.
- **Robust Logging**: Winston-based structured logging.

## Setup
1. `cd backend`
2. `npm install`
3. `npm start` (Runs on Port 5006)
4. Open `frontend/index.html`
