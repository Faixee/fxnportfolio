# ML Analytics Engine - Enterprise Edition

A high-performance predictive analytics platform featuring real-time time-series forecasting and feature importance analysis.

## Features
- **Predictive Modeling**: Simulated XGBoost inference engine.
- **Interactive Visualization**: Chart.js integration for trend analysis.
- **Explainable AI (XAI)**: Feature importance breakdown.
- **Low Latency**: Optimized API response times.

## Setup
1. `cd backend`
2. `npm install`
3. `npm start` (Runs on Port 5001)
4. Open `frontend/index.html`
