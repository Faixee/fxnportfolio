# IDP System - Enterprise Edition

An advanced Intelligent Document Processing platform leveraging Deep Learning (CNN/LSTM/BERT) simulation for high-accuracy extraction.

## Features
- **Neural Processing Pipeline**: Simulates Pre-processing, OCR, and NER stages.
- **Enterprise Security**: Helmet-hardened headers and rate limiting.
- **Real-time Feedback**: WebSocket-style stage updates (simulated).
- **Scalable Backend**: Node.js/Express with Clean Architecture.

## Setup
1. `cd backend`
2. `npm install`
3. `npm start` (Runs on Port 5005)
4. Open `frontend/index.html`
